# Todo Menubar

macOS 菜单栏待办应用 — 常驻顶部导航栏，支持手动输入和截图 AI 识别待办事项。

---

## 安装

1. 双击 `Todo Menubar_0.2.0_aarch64.dmg`
2. 将 `Todo Menubar.app` 拖入 `/Applications`
3. 首次打开：右键 → 打开（绕过 macOS 未签名拦截）

## 启动与退出

- 启动后应用常驻菜单栏，不会出现在 Dock 和 Cmd+Tab 中
- **左键点击**菜单栏图标 → 打开/关闭面板
- **右键点击**菜单栏图标 → 弹出菜单 → 点击 "Quit Todo Menubar" 退出
- **全局快捷键** `Cmd+Shift+T` → 任意位置唤起/隐藏面板

## 功能说明

### 1. 添加待办

- 在顶部输入框输入内容，按 **Enter** 或点击 **+** 按钮添加
- 左侧彩色圆点可切换优先级：🟢 低 → 🟠 中 → 🔴 高

### 2. 多行粘贴识别

- 复制多行文字，粘贴到输入框
- 自动检测多行内容，弹出确认面板
- 可逐条删除不需要的行，点击 "Add X tasks" 批量添加

### 3. 任务状态管理

| 状态 | 含义 | 操作 |
|------|------|------|
| **Ongoing** | 进行中（默认） | 新建任务自动为此状态 |
| **Pending** | 暂停/等待 | 悬浮任务 → 点击 ⏸ 暂停按钮 |
| **Done** | 已完成 | 点击左侧 checkbox 勾选 |

- Pending 状态点击 ▶ 可恢复为 Ongoing
- Done 状态再次点击 checkbox 可恢复为 Ongoing
- 顶部筛选栏：All / Ongoing / Pending / Done

### 4. 编辑任务

- **编辑标题**：悬浮任务 → 点击 ✎ 图标，或双击标题
- **编辑后** Enter 保存，Esc 取消
- **详情描述**：单击任务 → 展开描述区域，输入文字备注
- **粘贴图片**：在描述区域 Cmd+V 粘贴图片，自动保存并显示缩略图

### 5. 优先级

- 每个任务有三级优先级：低（绿）/ 中（橙）/ 高（红）
- 添加时可选，添加后点击彩色圆点切换
- checkbox 边框颜色随优先级变化

### 6. 截止日期

- 点击任务下方日历图标 📅
- 快速选择：Today / Tomorrow / Next Week / Clear
- 到期显示 "Today"、"Tomorrow"，过期显示红色 "Overdue"

### 7. 项目分类

- 顶部项目栏：All / Inbox / 自定义项目
- 点击 **+** 创建新项目（选颜色 + 输入名称）
- 选中项目后，新增任务自动归属该项目
- 悬浮项目标签 → 红色 × 删除项目（任务保留，归入 All）

### 8. 搜索

- 点击右上角 🔍 图标展开搜索框
- 实时过滤，匹配任务标题

### 9. 截图 OCR 识别

- 点击右上角 📷 图标进入截图模式
- **方式一**：Cmd+V 粘贴剪贴板中的截图（推荐）
- **方式二**：点击 "Select image file" 从文件选择
- AI 自动提取待办事项，显示勾选确认列表
- 可选择添加到哪个项目
- 支持中英文识别

**OCR 引擎：**

| 条件 | 引擎 | 说明 |
|------|------|------|
| 已设置 Claude API Key | Claude AI | 智能语义提取，准确率高 |
| 未设置 / 离线 | Apple Vision | 本地 OCR，基础文字识别 |

设置 API Key：点击右上角 ⚙ → 输入 Claude API Key → Save

### 10. 拖拽排序

- 按住任务拖动可调整顺序
- 松开后自动保存新位置

### 11. 到期提醒

- 设有截止日期且为今天的任务，自动发送 macOS 原生通知
- 每 5 分钟检查一次，每个任务只提醒一次

### 12. 月度自动归档

- 每月 1 日自动归档上月所有 Done 任务
- 归档基于任务完成时间（completed_at），非创建时间
- Ongoing / Pending 任务不会被归档
- 点击右上角 🗃 图标查看归档，按月份浏览
- 应用启动时 + 每小时自动检查归档

### 13. 创建时间

- 每个任务右侧显示创建日期（YYYY-MM-DD）
- 任务按创建时间先后排序

## 数据存储

- 数据库：`~/Library/Application Support/com.todo-menubar.app/todos.db`
- 图片附件：`~/Library/Application Support/com.todo-menubar.app/images/`
- 所有数据存储在本地，不上传任何服务器

## 技术栈

| 层 | 技术 | 版本 |
|----|------|------|
| 框架 | Tauri | v2.10 |
| 前端 | React + TypeScript + Vite | React 19 / Vite 5 |
| 后端 | Rust | 1.94.1 |
| 数据库 | SQLite | tauri-plugin-sql v2.4 |
| 菜单栏 | tauri-nspanel | v2.1 |
| OCR | Claude API + Apple Vision | — |
